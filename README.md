# ClawHalla.xyz

Landing page and documentation for [ClawHalla](https://github.com/deegalabs/clawhalla).

**Live:** [clawhalla.xyz](https://clawhalla.xyz)

## Stack

- [Astro](https://astro.build) 5.x
- [Starlight](https://starlight.astro.build) (docs)
- [Tailwind CSS](https://tailwindcss.com) 3.x

## Development

```bash
pnpm install
pnpm dev
```

## Pages

- `/` — Landing page (Hero, How It Works, Features, Agent Showcase, MC Preview, Pricing, Tech Stack)
- `/pricing` — Detailed pricing with FAQ
- `/changelog` — Version history v0.1 to v1.0
- `/community` — Links and contribution info
- `/docs/` — Full documentation (Starlight)

## Documentation

- Getting Started (Installation, Configuration, Claude Max, Server Setup)
- Guides (First Agent, Customization, Content Pipeline, Autopilot)
- Reference (Environment, Docker, API Reference, Architecture)
