ClawHalla is a Docker-based launcher for [OpenClaw](https://openclaw.ai) that removes all setup friction. Get your AI agent running in under 5 minutes.

## Why ClawHalla?

OpenClaw is a powerful AI agent framework, but setting it up requires:

- Node.js installation
- pnpm configuration
- Multiple CLI commands
- Understanding the ecosystem

ClawHalla wraps all of this in a pre-configured Docker container with helper scripts.

## Quick Start

```bash
curl -fsSL https://clawhalla.xyz/install.sh | bash
```

Or manually:

```bash
git clone https://github.com/deegalabs/clawhalla.git
cd clawhalla
cp .env.example .env
# Edit .env and add your ANTHROPIC_API_KEY
bash scripts/start.sh
docker compose exec clawhalla bash
openclaw onboard
```
