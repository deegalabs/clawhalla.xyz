## Environment file

Copy `.env.example` to `.env` and set the values you need.

Common keys:

- `ANTHROPIC_API_KEY`
- `OPENAI_API_KEY`
- `OPENCLAW_GATEWAY_TOKEN`

## Onboarding

Run interactive onboarding:

```bash
openclaw onboard
```

For automation, use non-interactive mode. See OpenClaw docs:

- [CLI Automation](https://docs.openclaw.ai/start/wizard-cli-automation)
- [onboard command](https://docs.openclaw.ai/cli/onboard)
