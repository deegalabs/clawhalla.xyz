---

## Steps

1. Start ClawHalla:

```bash
bash scripts/start.sh
```

2. Enter the container:

```bash
docker compose exec clawhalla bash
```

3. Run onboarding:

```bash
openclaw onboard
```

4. Open dashboard:

```bash
openclaw dashboard
```
